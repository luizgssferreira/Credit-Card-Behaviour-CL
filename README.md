# Understanding Customer Spending: A Data-Driven Credit Card Behavior Analysis

## Context 

Understanding credit card spending behavior is a key challenge in the financial industry. This involves analyzing various aspects of card usage, such as balances, purchases, cash advances, credit limits, and payments. While credit card usage patterns may appear random, deeper analysis reveals behavioral trends that can help identify and classify customer profiles.

This project is based on a real-world dataset containing detailed information from 9,000 active credit card accounts over six months. The dataset includes 18 behavioral variables that describe card usage patterns. The goal is to support the marketing team in designing tailored strategies for distinct customer groups by uncovering meaningful insights through advanced analysis.

Key objectives include:

Creating enriched customer profiles by deriving new Key Performance Indicators (KPIs) from the dataset.
Segmenting customers into distinct groups based on their credit card usage behaviors.
Identifying and interpreting the defining characteristics of each cluster.
Delivering actionable insights and recommendations for targeted marketing efforts.
